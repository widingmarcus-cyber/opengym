# Challenge 001: Fix the Syntax Error

## Difficulty: Easy

## Task

The file `setup/calculator.py` contains a simple calculator module, but it has syntax errors that prevent it from running.

**Your job:** Fix all syntax errors so the module works correctly.

## Rules

- Only modify files in the `setup/` directory
- Do not change the function signatures (names, parameters)
- The functions should work as their names suggest

## Expected Behavior

```python
from calculator import add, subtract, multiply, divide

add(2, 3)        # returns 5
subtract(10, 4)  # returns 6
multiply(3, 7)   # returns 21
divide(10, 2)    # returns 5.0
divide(10, 0)    # returns None
```
