# Challenge 115: Retry and Adapt

Write a data processor in `setup/solution.py`. Your first approach might not work -- if it fails, you'll get a detailed failure report to help you adapt.

This is a **multi-session challenge**. Read the step files in `steps/` for instructions:

1. **Step 1** (`steps/step_1.md`): Implement the initial solution
2. **Step 2** (`steps/step_2.md`): Review failure report and fix edge cases

Requirements are documented in `setup/requirements.md`.
