# Challenge 110: Undocumented Tool

## Objective

A data processing tool is available at `tools/mystery.py`. It has no documentation. Figure out how to use it to process `setup/input.txt` and produce `setup/output.txt`.

Hint: the tool reads from stdin and writes to stdout.

## Rules

- Do NOT modify any files in `tools/`.
- Write your result to `setup/output.txt`.
- The output must match the tool's transformation exactly.
