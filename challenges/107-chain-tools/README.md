# Challenge 107: Chain Tools

## Objective

Process the data in `setup/raw_data.txt` into a summary report at `setup/report.json`.

Three tools are available in `tools/`. Each tool's `--help` explains its input/output format. You need to figure out the correct order to chain them.

## Rules

- Do NOT modify any files in `tools/`.
- Write your final result to `setup/report.json`.
- The report must contain aggregated data per product with metadata.
