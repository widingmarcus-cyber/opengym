# Challenge 007: Find in the Docs

## Difficulty: Medium

## Task

The file `setup/documentation.md` contains the technical documentation for a fictional API called "WidgetAPI". It's long and detailed.

Your job: Read the documentation and fill in the answers in `setup/answers.py`.

Each function should return the exact answer as specified. Read the docs carefully.

## Rules

- Only modify `setup/answers.py`
- All answers can be found in `setup/documentation.md`
- Return exact strings/values — no extra whitespace, no paraphrasing
