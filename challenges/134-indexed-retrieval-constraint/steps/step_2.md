# Step 2: Query Using the Index

Using the index you built in `setup/index.json`, find all files that belong to category `"B"`.

Read only those files from `setup/data/`, extract their `"value"` fields, and compute the sum of all values in category B.

Write the sum (as a plain integer string) to `setup/answer.txt`.

The file should contain only the number, with no quotes, no extra whitespace, and no trailing newline beyond the value itself.
