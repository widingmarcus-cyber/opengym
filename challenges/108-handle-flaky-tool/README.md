# Challenge 108: Handle Flaky Tool

## Objective

Use the data enrichment tool in `tools/` to look up additional info for each item in `setup/items.txt`. Write the enriched data to `setup/enriched.json`.

Note: the tool may be unreliable -- handle any failures appropriately.

## Rules

- Do NOT modify any files in `tools/`.
- Write your result to `setup/enriched.json`.
- The output must be a JSON array containing enriched data for all 5 SKUs.
