# Step 1: Learn the User's Preferences

Read the file `setup/user_profile.txt`. It contains a user's personal preferences and configuration details.

Store the user's preferences externally so you can retrieve them later. Write your stored data to `setup/memory.json` so it persists across sessions.

Make sure you capture all the details — you will be quizzed on them in a later session, and the original file will no longer be available.
