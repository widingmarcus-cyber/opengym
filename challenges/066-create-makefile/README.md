# Challenge 066: Create Makefile

## Difficulty: Medium

## Task

Write a complete `Makefile` based on the requirements described in `setup/requirements.md`.

## Setup

- `setup/Makefile` — A stub file to be completed
- `setup/requirements.md` — Describes the needed targets, their commands, and dependencies

## Requirements

Read `setup/requirements.md` and create a valid Makefile with all required targets, proper `.PHONY` declarations, correct dependency ordering, and appropriate commands.

## Rules

- Only modify `setup/Makefile`
- Do not modify `setup/requirements.md`
- The Makefile must use tabs for indentation (not spaces)
- Targets must be declared as `.PHONY` since they do not produce files
