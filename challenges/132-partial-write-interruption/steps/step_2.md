# Step 2: Count Valid Records

Read `setup/data.json` and count the total number of valid records.

A valid record has:
- An `"id"` field (integer)
- A `"name"` field (string)
- A `"value"` field (string) that is NOT truncated (must contain "temperature", "humidity", and "pressure")

Write the count of valid records to `setup/answer.txt`. Write only the number, nothing else.
