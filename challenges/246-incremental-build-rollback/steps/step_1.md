# Step 1: Implement Module A

Implement `module_a` in `setup/module_a.py`.

## Requirements

Create a function `sort_records(records)` that:
- Takes a list of record dicts (each with "name" and "priority" fields)
- Sorts by "priority" **descending** (highest first)
- For records with equal priority, sorts by "name" **ascending** (alphabetical)
- Returns the sorted list

## Versioning

Save a versioned copy of your implementation as `setup/module_a_v1.py`.
