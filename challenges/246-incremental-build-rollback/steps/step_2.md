# Step 2: Implement Module B

Implement `module_b` in `setup/module_b.py`.

## Requirements

Create a function `filter_records(records, min_priority)` that:
- Takes a list of record dicts and a numeric min_priority threshold
- Keeps records where `priority >= min_priority` (greater-than-or-equal)
- Returns the filtered list
