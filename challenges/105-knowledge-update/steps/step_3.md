# Step 3: Recall Updated Knowledge

Using ONLY your stored data in `setup/memory.json`, answer these questions.

Write your answers to `setup/answers.json` in this exact format:

```json
{
  "alice_office": "Building ?, Room ???",
  "carol_role": "???",
  "dave_status": "active or departed",
  "iris_extension": "????",
  "bob_office": "Building ?, Room ???",
  "total_active_members": N
}
```

- `alice_office`: Alice's current office location
- `carol_role`: Carol's current role/title
- `dave_status`: Whether Dave is "active" or "departed"
- `iris_extension`: Iris's phone extension (as a string)
- `bob_office`: Bob's current office location
- `total_active_members`: Total number of currently active team members (integer)
