# Challenge 105: Knowledge Update

**Difficulty:** Hard
**Category:** memory-updating
**Dimension:** memory
**Type:** Multi-session (3 steps)

## Objective

Learn a team directory, then apply updates (moves, promotions, departures, new hires), then recall the UPDATED state — not the originals.

## What This Tests

- Ability to store structured information across sessions
- Correctly applying updates to previously stored data
- Distinguishing between original and updated values
- Handling additions and removals in stored knowledge

## Sessions

1. **Learn** — Read and memorize the team directory (8 members with roles, offices, extensions)
2. **Update** — Apply changes: a move, a promotion, a departure, and a new hire
3. **Recall** — Answer specific questions using only the updated memory

## Key Challenge

The agent must return UPDATED values, not the originals. This tests whether the agent truly modifies its stored knowledge or just appends new information.
