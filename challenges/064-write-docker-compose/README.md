# Challenge 064: Write Docker Compose

## Difficulty: Medium

## Task

Write a complete `docker-compose.yml` file based on the requirements described in `setup/requirements.md`.

## Setup

- `setup/docker-compose.yml` — A stub file to be completed
- `setup/requirements.md` — Describes the services, ports, environment variables, volumes, and networking requirements

## Requirements

Read `setup/requirements.md` carefully and create a valid Docker Compose configuration that satisfies all described requirements for the three services (web, database, cache).

## Rules

- Only modify `setup/docker-compose.yml`
- Do not modify `setup/requirements.md`
- The resulting file must be valid YAML and follow Docker Compose syntax
