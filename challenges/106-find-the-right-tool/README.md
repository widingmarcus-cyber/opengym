# Challenge 106: Find the Right Tool

## Objective

You need to convert the data in `setup/input.csv` to JSON format and write it to `setup/output.json`.

Tools are available in the `tools/` directory. Figure out which one to use.

## Rules

- Do NOT modify any files in `tools/`.
- Write your result to `setup/output.json`.
- The output must be valid JSON containing all records from the CSV.
