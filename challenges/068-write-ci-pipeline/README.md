# Challenge 068: Write CI Pipeline

## Difficulty: Hard

## Task

Write a complete GitHub Actions CI/CD pipeline based on the requirements described in `setup/requirements.md`.

## Setup

- `setup/pipeline.yml` — A stub file to be completed
- `setup/requirements.md` — Describes the pipeline requirements in detail

## Requirements

Read `setup/requirements.md` and create a valid GitHub Actions workflow with a build matrix, dependency caching, test execution, conditional deployment, and artifact uploads.

## Rules

- Only modify `setup/pipeline.yml`
- Do not modify `setup/requirements.md`
- The resulting file must be valid YAML and follow GitHub Actions workflow syntax
